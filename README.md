# Weather App 
An application ran with comandline prompts, to diplay typescript functions and methods. Shows Interfaces also.

## Installations 
Uses node package modules
Inquirer  
Express  
Day.js  
Vite

## Usage 
node.js   
npm install inquirer  
npm install  
npm run start:dev
